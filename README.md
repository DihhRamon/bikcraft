# 🚲 Bikcraft — Projeto Front-end

🔗 **Live Demo:**  
https://dihhramon.github.io/bikcraft/

---

## 💼 Sobre o projeto

O **Bikcraft** é um projeto front-end desenvolvido com foco em boas práticas de desenvolvimento web moderno. Trata-se de um site institucional para uma marca fictícia de bicicletas personalizadas, com ênfase em design, usabilidade e responsividade.

Este projeto faz parte do meu portfólio e demonstra minha capacidade de transformar layouts em interfaces funcionais, organizadas e responsivas.

---

## 🎯 Objetivos do projeto

- Desenvolver um layout moderno e profissional  
- Praticar HTML semântico e acessível  
- Criar estilos reutilizáveis e escaláveis com CSS  
- Implementar responsividade completa (mobile-first)  
- Aplicar interações básicas com JavaScript  

---

## 🧠 Principais aprendizados

- Estruturação de páginas multi-seção  
- Organização de código CSS (modularização e reutilização)  
- Técnicas de responsividade com media queries  
- Melhoria da experiência do usuário (UX/UI)  
- Boas práticas de desenvolvimento front-end  

---

## 🛠️ Tecnologias utilizadas

- **HTML5** — Estrutura semântica  
- **CSS3** — Estilização e responsividade  
- **JavaScript** — Interatividade básica  

---

## 📱 Responsividade

O projeto foi desenvolvido com abordagem **mobile-first**, garantindo compatibilidade com:

- 📱 Smartphones  
- 💻 Tablets  
- 🖥️ Desktops  

---

## 🖼️ Funcionalidades

- Página inicial com destaque de produtos  
- Seção de benefícios da marca  
- Página de produtos  
- Página de seguros  
- Formulário de contato  
- Layout moderno e intuitivo
